Step 2 · Visualise
Canopy
// sitemap toolkit

Crawl any site and Canopy maps every internal URL into an interactive tree. Explore by section and depth, collapse branches, click nodes for detail. Export the diagram as a PNG or SVG — useful for presentations, proposals, or understanding a site before a redesign.

🕷️ Live crawler 🌳 Tree diagram ↕ Depth control 🎨 Colour themes 🖼️ PNG / SVG export 📋 CSV / JSON

Step 3 · Migrate
Waypoint
// site migration toolkit

Paste in your old and new URL lists and Waypoint cross-references them — suggesting the best redirect for every changed URL, flagging loops, chains, and mergers, and letting you confirm or adjust each one. Export in CSV, .htaccess, or Nginx format when you're done.

🗺️ URL mapper 🧪 Loop + chain detection 🕷️ Built-in spider 📊 Validator ↓ .htaccess / Nginx / CSV 🔗 Shareable sessions
