Step 4 · Inventory
Atlas
// content inventory

Atlas turns the crawl into a working content inventory. Every page gets a disposition — keep, rewrite, merge, redirect, or delete — plus auto-raised flags for missing titles, thin content, and short meta descriptions. When you're ready, send redirect-flagged URLs straight to Waypoint.

📋 Full page inventory 🚩 Auto-flag system ✏️ Dispositions 📊 Word count + readability 📝 Per-page notes ↺ Recrawl for metadata ↓ CSV export

Forge
// IA designer

Step 6 · Redirects
Waypoint
// site migration toolkit

Paste in your old and new URL lists and Waypoint cross-references them — suggesting the best redirect for every changed URL, flagging loops, chains, and mergers, and letting you confirm or adjust each one. Export in CSV, .htaccess, or Nginx format when done.

🗺️ URL mapper 🧪 Loop + chain detection 🕷️ Built-in spider 📊 Validator 📝 Per-row notes ↓ .htaccess / Nginx / CSV ⚡ Diff tool
