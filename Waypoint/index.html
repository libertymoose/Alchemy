      What is Waypoint?
      
    

    

      
Waypoint is a site migration toolkit.

      
Plan, map, and validate every URL redirect before a site moves, relaunches, or restructures. No installs, no spreadsheets. Everything runs in the browser.


      
The workflow

      
🗺️ Mapper — Cross-reference old and new URLs, confirm redirect destinations, and manage your full migration map.

      
🧪 Analysis — Checks the logic of your redirect map. Catches circular redirects, unnecessary chains, and destinations with multiple competing sources.

      
🕷️ Spider — Crawls a live site to discover all its internal URLs, then sends them straight to the Mapper.

      
📊 Validator — Final QA pass. Checks for issues, generates a shareable report, and exports your redirect map in CSV, .htaccess, or Nginx format.

Privacy

      
Waypoint runs entirely in your browser. No URLs, session data, or redirect mappings are transmitted to or stored on any server — with two exceptions. The Spider's proxy feature routes page requests through a Cloudflare Worker to bypass browser security restrictions. The share link feature stores session data in a Cloudflare Worker so it can be retrieved via a short link. If you're not using either of these features, Waypoint is completely local.

Alchemy is a suite of browser-based tools for website discovery, auditing, and migration. No installs, no logins, no spreadsheets. Each tool does one thing well — together they cover the whole journey.
