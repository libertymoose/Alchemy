What is Scout?

Scout is a website audit toolkit.

Point it at any URL and Scout runs eight automated checks in parallel — Lighthouse scores, security headers, sitemap analysis, tech stack detection, DNS, accessibility, GDPR compliance, and readability. Add your observations to each section, then export a polished client-ready report.

The checks

⚡ PageSpeed — Desktop and mobile Lighthouse scores, Core Web Vitals, third-party impact, and SEO diagnostics.

🔒 Security — Six response headers, HTTPS enforcement, SPF and DMARC records, and admin exposure checks.

🗺️ Sitemap + Robots — Page count, section and depth breakdown, content freshness from lastmod, robots.txt validation.

🔌 Trackers + Accessibility — Third-party scripts, GDPR checklist, Lighthouse a11y score, heading structure, alt text coverage, and form label analysis.

✦ Part of the Alchemy suite
Boring legal stuff
Boring stuff

Scout uses the Google PageSpeed Insights API (authenticated via Cloudflare Worker) for performance data. Security headers, sitemap, robots.txt, tracker detection, and readability checks all run via the Scout proxy Worker. Nothing is fetched directly from the page — all automated checks go through Cloudflare.

Audits are saved as .json files. Nothing is sent to a server except the check API calls. AI draft summaries are routed through a Cloudflare Worker — the API key is stored securely as a Worker secret and is never exposed in the page.

✦ Alchemy suite

Alchemy is a suite of browser-based tools covering the full website production and migration pipeline. No installs, no logins, no spreadsheets. Each tool does one thing well — together they cover the whole journey.

Live tools

🔍 Scout — Website audit tool. Eight automated checks: Lighthouse scores, security headers, sitemap analysis, tech stack, DNS, accessibility, GDPR compliance, and readability. → Open Scout

🌿 Canopy — Site structure mapper. Crawls a live website, maps every URL into an interactive tree diagram, and exports as PNG, SVG, CSV, or JSON. → Open Canopy

🧭 Waypoint — Redirect mapping toolkit. Cross-references old and new URLs, catches loops and chains, exports in CSV, .htaccess, or Nginx format. → Open Waypoint

In development

🏗 Alchemy — Project setup and shared crawl. Create a project, run the crawl once, and every tool reads from the same dataset.

🗺 Atlas — Content inventory. Catalogue every page with metadata and assign a disposition — keep, rewrite, merge, redirect, or delete.

🔨 Forge — IA designer. Design the new site structure from scratch or by importing the existing one. Exports a URL list straight into Waypoint.

🛡 Warden — Launch validation. Pre and post-launch checks — redirects resolving, new pages live, no 404s.

📋 Charter — Project roadmap planner. Imports findings from any tool at any stage and generates a prioritised, hour-budgeted roadmap with a client-facing export.
