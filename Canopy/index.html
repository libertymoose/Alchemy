What is Canopy?

Canopy is a sitemap toolkit.

Crawl any live website and Canopy maps every reachable internal URL into an interactive tree diagram. Explore the architecture by section and depth, then export the structure in whatever format you need.

The workflow

🕷️ Crawler — Point it at any URL and Canopy walks all reachable internal pages. Use the Workbench to filter out what you don't need.

🌳 Diagram — An interactive tree laid out by section and depth. Pan, zoom, and click any node to inspect it. Collapse branches to focus on what matters.

🍂 Export — PNG or SVG for presentations, CSV for spreadsheets, JSON for developers.

Privacy

Canopy runs entirely in your browser. The only exception is the proxy feature: when crawling a site, page requests are routed through a Cloudflare Worker so the browser can reach external sites. Only the URLs being fetched are passed through.
